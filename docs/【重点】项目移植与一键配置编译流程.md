# 当前工程项目移植与一键配置编译流程

## 1. 新建项目

```bash
cd /data/tr531x_proj
cp -a projects/demo_sle_uart projects/<new_project>
```

至少修改：

1. `projects/<new_project>/sdk_overlay/custom/*`
2. `projects/<new_project>/config/standard_tr5310_s.config`
3. `projects/<new_project>/scripts/post_build.sh`（可选）

## 2. 配置默认编译参数

```bash
./tools/tr531x/config_build_profile.sh
```

写入：`tools/tr531x/build_profile.conf`

## 3. 一键编译

```bash
./tools/tr531x/build_project.sh
```

## 4. menuconfig（项目上下文）

```bash
./tools/tr531x/project_menuconfig.sh --project <new_project> --target standard-tr5310-s --patch-set default
```

## 5. 固化配置变化到 SDK patch（可选）

```bash
./tools/tr531x/project_menuconfig.sh --project <new_project> --target standard-tr5310-s --patch-set default --save-to-sdk-patch --patch-note "enable xxx"
```
