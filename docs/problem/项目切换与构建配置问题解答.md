# 项目切换与构建配置问题解答

## 1. 如何切换项目

推荐直接用：

```bash
./tools/tr531x/build_project.sh --project <name> --target standard-tr5310-s --patch-set default
```

脚本会自动：

1. 应用 patch；
2. 切换 `application/samples/custom`；
3. 注入项目 config；
4. 编译并导出产物；
5. 恢复现场。

## 2. 参数不想每次输入

先执行：

```bash
./tools/tr531x/config_build_profile.sh
```

后续直接：

```bash
./tools/tr531x/build_project.sh
```

## 3. SDK 改动如何管理

统一放到：

`patches/sdk/sdk_root_dir/<patch_set>/`

应用：

```bash
./tools/tr531x/apply_sdk_patches.sh <patch_set>
```
