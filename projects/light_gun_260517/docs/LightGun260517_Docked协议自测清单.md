# LightGun260517 Docked 协议自测清单

## 1. 文档目的

- 用于联调/回归/产测时，对 `projects/light_gun_260517` 的 Docked 协议做逐条验证。
- 所有命令都基于当前实现的 `proto_docked.c`。

## 2. 测试准备

1. 固件构建
- 光枪端：`./tools/tr531x/build_project.sh --project light_gun_260517`
- Dongle 端：`./tools/tr531x/build_project.sh --project light_gun_dongle_260517`

2. 连接方式
- 有线：USB CDC（串口语义 512000）
- 无线：SLE 路由后按同样字节流命令测试

3. 报文格式
- 本文默认命令输入为十六进制字节序列。
- 响应中的 `rc=0` 代表成功，`rc!=0` 代表失败。

## 3. 基础与模式命令

1. 握手
- 输入：`01 02`
- 期望响应：`81 02 00`
- 失败判据：无响应，或首字节不是 `81`

2. 设置模式
- 输入：`10 <mode>`，示例 `10 00`
- 期望响应：`90 <mode> 00`
- 失败判据：返回 mode 与输入不一致

3. 查询模式
- 输入：`11`
- 期望响应：`91 <mode>`
- 失败判据：无响应，或首字节不是 `91`

## 4. 通用配置命令

1. 写配置字节
- 输入：`20 <idx> <val>`，示例 `20 03 55`
- 期望响应：`A0 <idx> <val> 00`
- 失败判据：返回 idx/val 不一致

2. 读配置字节
- 输入：`21 <idx>`，示例 `21 03`
- 期望响应：`A1 <idx> <val>`
- 失败判据：与上一步写入值不一致

## 5. 外设控制与状态命令

1. LED 控制
- 输入：`31 <v>`，示例 `31 01`
- 期望响应：`B1 <v> 00`
- 失败判据：无响应或 rc 非 0

2. 反馈控制
- 输入：`32 <v>`（建议 `0~100`）
- 期望响应：`B2 <v> 00`
- 失败判据：无响应或 rc 非 0

3. 读取存储块
- 输入：`33`
- 期望响应：`B3 ...`
- 失败判据：首字节不是 `B3`

4. 查询自检/温度/链路
- 输入：`34`
- 期望响应：`B4 <selftest_bitmap[4]> <temp[2]> <transport_type>`
- 说明：`transport_type` 期望值
  - `0`：USB CDC
  - `1`：SLE
- 失败判据：长度不足 8 字节，或首字节不是 `B4`

## 6. 绑定命令（svc_binding）

1. 读取绑定块
- 输入：`40`
- 期望响应：`C0 ...`
- 失败判据：首字节不是 `C0`

2. 设置绑定项
- 输入：`41 <slot> <code>`，示例 `41 00 01`
- 期望响应：`C1 <slot> <code> <rc>`
- 成功判据：`rc=00`
- 失败判据：`rc!=00` 或回显不一致

3. 保存绑定
- 输入：`42`
- 期望响应：`C2 <rc>`
- 成功判据：`rc=00`
- 失败判据：`rc!=00`

## 7. 校准命令（svc_calibration）

1. 进入校准
- 输入：`50`
- 期望响应：`D0 <rc>`
- 成功判据：`rc=00`

2. 推入采样
- 输入：`51 <x_l> <x_h> <y_l> <y_h>`
- 示例：`51 20 03 58 02`（x=0x0320, y=0x0258）
- 期望响应：`D1 <rc> <state>`
- 说明：累计足够采样后 `state` 会进入完成态

3. 提交校准
- 输入：`52`
- 期望响应：`D2 <rc> <cx_l> <cx_h> <cy_l> <cy_h>`
- 成功判据：`rc=00`

4. 查询校准状态
- 输入：`53`
- 期望响应：`D3 <state> <cx_l> <cx_h> <cy_l> <cy_h>`
- 失败判据：首字节不是 `D3`

## 8. 主动上报帧检查

1. 帧头
- 期望：`E0`

2. 关键字段
- `mapped_keys`：按键经过绑定映射后的值
- `adc0/adc1`：ADC 原始采样
- `ir_x/ir_y`：已应用校准偏移的坐标
- `cal_state`：校准状态
- `key_map0`：绑定槽 0 的当前映射值
- `mode`：当前模式

3. 一致性检查
- 同一业务动作下，有线 USB 与无线 SLE 上报字段语义一致。

## 9. 回归判定标准

1. 必过项
- 第 3、4、6、7 章命令全部有响应且成功项 `rc=00`。
- 主动上报帧可持续输出，字段位置稳定。

2. 可降级项
- 温度字段异常但协议其他项正常，可记为 `WARN`。

3. 失败项
- 命令无响应、响应头错误、字段错位、反复卡在 FAULT 不恢复。
